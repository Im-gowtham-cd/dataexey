Dataexey.io

The official browser extension prevents duplicate file downloads by detecting files that already exist. It saves storage, reduces bandwidth usage, and keeps your downloads organized.

Version

Guideline for setup

    1) Download and extract the Dataexey browser extension.

    2) After extraction, open the \download_files folder.

    3) Locate and open the newdetect.py file.

    4) On line 8, change the path from r"C:\Users\Admin\Downloads" to r"C:\Users\your-name\needed-folder".

    5) Open Google Chrome and go to chrome://extensions.

    6) Turn on Developer Mode in the top right corner.

    7) Click the Load Unpacked button.

    8) Select the folder path Duplicate-detection\duplicate-download-detector.

    9) Upload it to complete the setup.

How to use ?

    1) Start the newdetect.py file in the terminal as ' python newdetect.py '

    2) Open Google Chrome and start downloading files.

    3) The extension will automatically detect duplicate files and prompt you to choose an action.

    4) You can choose to keep the new file, replace the old file, or cancel the download.

    5) The extension will continue running in the background, monitoring your downloads for duplicates.

    6) Enjoy a clutter-free download experience with Dataexey!

    7) If you encounter any issues or have suggestions for improvement, please don't hesitate to contact us.

    8) Thank you for using Dataexey!

    9) For more information, visit our official website or check out our GitHub repository.

About

Dataexey is a lightweight and efficient browser extension designed to prevent duplicate file downloads. By detecting files that already exist in your system, it saves storage space, reduces bandwidth usage, and keeps your downloads organized. Our mission is to simplify file management and provide users with a clutter-free browsing experience.
