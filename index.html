Dataexey.io

The official browser extension prevents duplicate file downloads by detecting files that already exist. It saves storage, reduces bandwidth usage, and keeps your downloads organized.

Version

Guideline for setup

    1) Download and extract the Dataexey browser extension.

    2) After extraction, open the \download_files folder.

    3) Locate and open the newdetect.py file.

    4) On line 8, change the path from r"C:\Users\Admin\Downloads" to r"C:\Users\your-name\needed-folder".

    5) Open Google Chrome and go to chrome://extensions.

    6) Turn on Developer Mode in the top right corner.

    7) Click the Load Unpacked button.

    8) Select the folder path Duplicate-detection\duplicate-download-detector.

    9) Upload it to complete the setup.
